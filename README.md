## Development Setup Local

1. Clone repository

```bash
git clone git@github.com:AlexGalhardo/contacts.alexgalhardo.com.git
```

2. Enter folder

```bash
cd contacts.alexgalhardo.com/
```

3. Install dependencies

```bash
bun install
```

4. Start local server

```bash
bun run dev
```

5. Go to: <http://localhost:3000>
